# PROJECT VISION: SHAADI MANTRANA

## MISSION
To create a trust-first, premium matrimony experience for urban Indian professionals, combining traditional values with modern mobile convenience.

## APP PHILOSOPHY
- **Trust over Growth**: Authenticity and safety are more important than sheer user volume.
- **Elegance over Flash**: The UI must feel luxurious and respectful, not like a generic dating app.
- **Mobile-First**: Every feature must be optimized for Capacitor/mobile usage.
